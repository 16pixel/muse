Installation

Use manually

Download the compiled and minified css file (core ~10KB min+gzip)

Download 

<link rel="stylesheet" href="./dist/muse.min.css">

Add muse via cdn

Alternatively, you can use the unpkg or cdnjs CDN to load compiled muse.css.

<link rel="stylesheet" href="https://unpkg.com/@16px/cherry/dist/muse.min.css">
We recommend our users to lock Muse's version when using CDN. Please refer to unpkg.com for more information.

Install muse via npm

npm i @16px/muse --save-dev
