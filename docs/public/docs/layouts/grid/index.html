Responsive

You can define a column size for each viewport size: mobile(xs), tablet(sm), and desktop(md~xl).

col-<1-12> apply to window width smaller than or equal to 480px.

col-sm-<1-12> apply to window width greater than or equal to 600px.

col-md-<1-12> apply to window width greater than or equal to 840px.

col-lg-<1-12> apply to window width greater than or equal to 960px.

col-xl-<1-12> apply to window width greater than or equal to 1280px.
